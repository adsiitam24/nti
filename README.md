# nti

p8Script1
Inserta 3*n registros en la tabla tagmessagesde la base taglogs. 
El valor default de n es 3, pero se puede elegir el número de conjunto de 3 registros que se quiere insertar.
También se puede cambiar el número de secuencia inicial para cada conjunto de 3 registros (seq), la hora y minutos por default (hora) 
y el segundo en el que empieza (seg). 

p8Script2
Actualiza el valor del campo location de los últimos n registros de la tabla tagmessagesde la base taglogs. 
El valor default de n es 3, se puede cambiar modificando el valor de la variable con el mismo nombre.
El valor default del valor que se asignará es "EFRA2", es posible cambiarlo antes de ejecutarlo.
